# js-filecoin-explorer

    - npm install
    - npm run build
    - npm start

- Code

        - Redux
        - Axios
        - Webpack
        - Less
        - Storybook (类似antd的前端UI框架)
        - amCharts (地图组件的组件库)
    - Backend
        - Nodejs
        - Express
        - Axios
        - Mongodb
